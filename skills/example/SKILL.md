# Example Skill

This skill exists as a reference template. Copy it to scaffold a new skill, then replace its content with your own instructions.

## When to use

Describe the specific scenarios where the agent should apply this skill. Be concrete — include the types of requests, keywords, or contexts that should trigger it.

## Instructions

2. Step two — continue with the next action.

## Notes

- Keep instructions concrete and action-oriented.
- Avoid duplicating knowledge the agent already has — only add what's domain-specific or non-obvious.
- If the skill requires external context (an API, a schema, a doc), link to it or include a snippet here.
